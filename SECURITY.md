# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Report security vulnerabilities to **security@ansvar.eu**.

- Do not open a public GitHub issue for security vulnerabilities.
- Include steps to reproduce, impact assessment, and any suggested fix.
- We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Security Measures

- All dependencies are scanned via GitHub Advanced Security (CodeQL, Dependabot, secret scanning).
- No secrets or credentials are stored in this repository.
- This MCP server is stateless — it does not store or process client data.
